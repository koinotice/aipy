Compile Locally

If we don't create builds for your platform, feel free to compile it locally

git clone git@github.com:tinytacoteam/zazu.git
cd zazu
npm install
npm run release
open releases

Install via Brew

For Macs you can also install the latest build using Homebrew

brew update
brew cask install zazu
